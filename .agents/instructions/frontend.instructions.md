# Frontend Instructions

Scope: React and TypeScript work under `src/frontend`.

- Organize behavior by feature and keep API access outside components.
- Model server data explicitly; show loading, empty, and error states.
- Use semantic HTML, labels, keyboard access, and visible focus.
- Prefer browser and React features over dependencies or abstractions.
- Test changed user behavior with mocked API boundaries.
- Run the repository's frontend test, type-check, and build scripts.
