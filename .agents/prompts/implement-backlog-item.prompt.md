# Implement Backlog Item

## Start

1. Read `AGENTS.md`, the target item, and only its referenced context.
2. Inspect current code and git status; preserve unrelated changes.
3. State a short plan and stop on the item's stop conditions.
4. Implement only the acceptance criteria.

## Validate

Run the listed commands plus the narrowest relevant tests. Review the diff for scope, safety, and missing coverage. If a check is skipped or fails, say why.
