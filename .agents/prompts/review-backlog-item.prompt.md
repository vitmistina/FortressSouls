# Review Backlog Item

## Start

1. Read `AGENTS.md`, the target item, relevant primitives, and accepted ADRs.
2. Inspect the diff and existing tests without changing files.
3. Map acceptance criteria to evidence.

## Validate

Run or inspect the narrowest relevant checks. Review correctness, read-only safety, security, boundaries, regressions, and test gaps.

## Summary

- Findings by severity with file locations
- Open questions or assumptions
- Validation run
- Acceptance criteria status
- Residual risk; say explicitly if there are no findings
